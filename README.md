# NoteShare Web

The web-based counterpart to the NoteShare ecosystem. This frontend application provides users with access to their shared notes directly from their browser, ensuring accessibility from any device.

## Features
- **Responsive Design**: Optimized for desktop, tablet, and mobile browsers.
- **Real-Time Updates**: View and share notes instantly.

## Tech Stack
- JavaScript (Frontend framework)
- HTML / CSS

## Getting Started
1. Clone the repository.
2. Navigate to the `frontend` directory.
3. Serve the static files using a local development server (e.g., Live Server or python HTTP server).
